# XMX Blog

## Descrição

Aplicação web de Blog para o desafio ténico da XMX Corp consumindo API externa para exibir posts, comentários e dados de usuários.

## Tecnologias Utilizadas

- Laravel 12.40.2

## Instalação

### Pré-requisitos

- PHP 8.2+

### Passos

1. Clone o repositório
2. Instale as dependências PHP
3. Configure o banco de dados
4. Execute as migrations
